# Jobman
Schweizer Job aggregator, welcher Daten von allen gängigen Job-Suchmaschinen der Schweiz zusammenführt und in verständlich lesbarer Form darstellt.

#Features
< Keyword search
< Canton Filter
< Branche Filter
< API + web frontent

#Quick start (Docker)
  1. Install Docker
  2. Run:
     docker-compose up --buiöd
